## Getting Started  
### Connect Your X Account  
```python  
from proxio import Client  

proxio = Client(api_key="YOUR_PROXIO_KEY")  
proxio.train_bot(  
    x_handle="@yourhandle",  
    persona="professional"  # or "casual", "humorous", etc.  
)  
```  

### Embed Your Chatbot  
```html  
<!-- Add to your website -->  
<script src="https://cdn.proxio.ai/embed.js" data-bot-id="YOUR_BOT_ID"></script>  
```
