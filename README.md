## Getting Started
### Create Your Profile
```json
{
    "interests": ["crypto", "AI", "blockchain"],
    "expertise": ["trading", "machine learning"],
    "connections": "public"
}
```

### API Integration
```python
from appit import AppitAPI

client = AppitAPI(api_key="YOUR_API_KEY")
feed = client.get_personalized_feed(
    topics=["crypto", "AI"],
    content_type="analysis"
)
```
